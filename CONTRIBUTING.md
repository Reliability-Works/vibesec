# Contributing

Thanks for contributing to VibeSec.

## Development

- Install: `pnpm install`
- Build: `pnpm build`
- Lint: `pnpm lint`
- Typecheck: `pnpm typecheck`
- Test: `pnpm test`

## Pull requests

- Keep changes focused and easy to review.
- Add/adjust tests when behavior changes.
- Do not add or commit secrets (including real `.env` files).

## Rule contributions

Rules are intended to be declarative by default. Avoid designs that require executing code from scanned repositories.
